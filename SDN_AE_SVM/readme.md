AE-SVM  
项目框架

├── AE_SVM.py               # 结合自编码器和SVM的模型  
├── train.py                # 模型训练脚本  
├── test.py                 # 模型测试和评估脚本  
├── utils.py                # 工具函数  
├── main.py                 # 主程序入口  
└── requirements.txt        # 项目依赖
